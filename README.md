# The Onion Article Scraper

## Overview
An app that scrapes news articles from the Onion's website, displays the articles, and lets users save and delete their articles. 

[Check out the App] (http://bit.ly/mongoscraper)

## Technologies Used

- Node.js
- Express.js
- Handlebars.js
- MongoDB
- Mongoose
- Cheerio
- Bootstrap
- jQuery
